# WebCrawler-BaiduTieba-master
## 使用方法
* 编辑URLs.txt</br>
加入帖子链接，以换行符分隔</br>
```
http://tieba.baidu.com/p/5541675366
http://tieba.baidu.com/p/5668450696
http://tieba.baidu.com/p/5652858725
http://tieba.baidu.com/p/5358795652
```
* 在命令行启动</br>
``` 
$ python2 myBaiduCrawler.py
```
